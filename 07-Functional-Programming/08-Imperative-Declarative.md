# Imperative vs Declarative

## Imperative

- What to do & How to do it.
- Computer
- jQuery
- ideas
  - easier to read
  - more productive

## Declarative

- What to do & what should happen.
- Humans
- React

```js
// Imperative (how to do)
for (let i = 0; i < 1000; i++) {
  console.log(i);
}

// Declarative (what to do)
[1, 2, 3].forEach(item => console.log(item));
```
